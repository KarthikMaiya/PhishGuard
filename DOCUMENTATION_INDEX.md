- **What:** `subdomain_count()` returned raw dot count instead of actual subdomains
- **Impact:** google.com incorrectly classified as HIGH-risk phishing
- **Fix:** Changed formula to `max(dot_count - 1, 0)`
- **Status:** FIXED and verified

- **Algorithm:** XGBoost binary classifier
